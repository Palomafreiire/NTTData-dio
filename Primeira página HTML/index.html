O que é HTML?

HTML (The HyperText Markup Language) ou Linguagem de Marcação de Hipertexto em português, é uma linguagem de marcação que compõe grande parte das páginas webs que utilizamos atualmente. Utiliza HTML, como o nome já diz, serve para marcar a pagina web, para posicionar a estutrura de toda a página.

Tipos de tags do HTML

  1. tags estruturais: mudam o posicionamento de elementos na web;
  2. tags de cabeçalho: estão relacionadas com o cabeçalho e alguns elementos básicos de sites, como o título e o favicon;
  3. tags de links: são utilizadas para gerenciar qualquer tipo de link existente na página;
  4. tags de listas: permitem criar diferentes listas (como esta!);
  5. tags de formatação de caracteres: adicionam formatação especial a qualquer texto exibido na página web;
  6. tags de manipulação de elementos: tornam mais preciso o posicionamento de conteúdos no site;
  7. tags de conteúdo multimídia: utilizadas para inserir elementos como imagens, vídeos e músicas nas páginas web;
  8. tags de frames: para criar frames (que funciona como uma divisão interna da página) com conteúdos internos;
  9. tags de formulários: utilizadas para criar formulários para os usuários;
  10. tags de tabelas: adotadas para criar e modificar tabelas facilmente.

HTML é uma Linguagem de programação?

Mesmo "linguagem" estando no nome do HTML (Linguagem de Marcação de Hipertexto), não é uma linguagem de programação propriamente dita, já que para ser uma linguagem de programação tem que criar funcionalidade dinâmicas e o HTML só faz a estrutura da página em si. No HTML o programador pode criar e estrutrurar seções, paragrados, links utilizando suas tags e atributos.
